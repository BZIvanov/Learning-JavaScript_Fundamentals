To be able to use ES6 modules with export/import we need to specify type='module' for the script tag.

Check the console for the printed message.

ES6 modules info

  • by specifying a file as module, everything in it will be private, not on the global scope.
  • modules always use strict mode.
  • 'this' will be undefined, not the global scope if used directly in the file.
  • module scripts are loaded with defer
